### Ce que je construis

Tout l'écosystème technique de **Blox Fruits FR**, en production et auto-hébergé :

- 🤖 **9 bots Discord** - économie, modération, PvP classé, raids PvE, premium, support
- 🌐 **Plateforme web** (`bffr.fr`) - transcripts, panels admin, casino, paiements (Stripe / PayPal / crypto)
- 💰 **Économie de jeu complète** - monnaie, battle pass, boutique, quêtes, crafting
- ⚔️ **Système de raids** - 30+ boss, fruits du démon, sets d'équipement, donjons, classements
- 🏗️ **Infrastructure** - Docker + PM2, PostgreSQL, Caddy, monitoring
